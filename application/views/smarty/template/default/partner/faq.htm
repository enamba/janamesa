Hilfe

  • Was kann ich hier tun?

    Das Partnerkonto ist eine Hilfe von {$config->domain->base} für unsere Dienstleister. Wir möchten damit Ihren Arbeitsablauf vereinfachen. Stellen Sie wichtige Sachen rund um Ihren Lieferservice schnell und sicher selbst ein! Im Partnerkonto können Sie:

    • Die Bestellungen sehen, die bei {$config->domain->base} für Ihren Lieferservice eingingen
    • Den von Ihnen erzielten Umsatz bei {$config->domain->base}
    • Die Rechnungen von {$config->domain->base} einsehen
    • Statistiken einsehen: wieviel Bestellungen gab es diesen Monat, und wieviele waren es letzten Monat?
    • Rabatte selbständig auf Ihre Menükarte einstellen
    • Einfach und schnell mit {$config->domain->base} in Kontakt treten
  • Wo sehe ich Bestellungen?

    Sie können folgende Bestellungen sehen:

    • Bestellungen von heute
    • Bestellungen der letzten 7 Tage
    • Bestellungen dieser Woche
    • Bestellungen diesen Monats
    • Bestellungen des letzten Monats

    Es gibt zwei Möglichkeiten, diese Bestellungen zu sehen:

    • Erstens können Sie jederzeit in der roten Kopfzeile auf jeder Seite einfach auf die Bestellungen klicken, die Sie sehen wollen
    • Zweitens können Sie auch auf den Menüpunkt Bestellungen klicken, um Ihre Bestellungen zu sehen
  • Wieviel Geld verdiene ich mit Lieferando? - Rechnungen

    Unter „Rechnungen“ sehen Sie genau, wieviel Umsatz Sie im aktuellen Monat mit {$config->domain->base} gemacht haben. Außerdem sehen Sie den monatlichen Umsatz für das aktuelle und das letzte Jahr. So wissen Sie genau, wieviel Geld Sie jeden Monat mit {$config->domain->base} eingenommen haben. Sie können sich auch alle vorhandenen Rechnungen ansehen und diese ausdrucken.

    Sie habe auch die Möglichkeit, sich zu entscheiden, wie {$config->domain->base} Ihnen die Rechnung zuschicken soll.

    • Entweder die Rechnung kommt kostenlos per E-Mail und per Partner-Konto
    • Oder wir faxen die Rechnung (das kostet 2,50 € pro Rechnung)
    • Oder wir schicken die Rechnung per Post (das kostet 5,00 € pro Rechnung).

    Unten auf der Seite finden Sie alle Rechnungen von {$config->domain->base} aus diesem und aus dem letzten Jahr.

  • Statistiken

    In den Statistiken sehen Sie Ihre aktuellen Verkaufszahlen für den aktuellen Monat, nach Tag sortiert. Sie sehen:

    • Die Anzahl der Bestellungen
    • Den insgesamt über lieferando erzielten Umsatz
    • Die Bestellungen, die online bezahlt wurden
    • Die Bestellungen, die bar bezahlt wurden

    In der zweiten Tabelle „Eigene Domain gesamt“ sehen Sie alle Bestellungen über Ihre eigene Webseite (sofern sie bei lieferando eine Seite verwalten. Möchten Sie, dass lieferando den Internet-Auftritt Ihres Lieferservices übernimmt? Dann klicken Sie hier.


    Im Graphen darunter sehen Sie die Umsätze und die Bestellungen jeder Kalenderwoche des aktuellen und des vergangenen Jahres. Unter den Graphen sehen Sie diese Zahl noch einmal in Tabellenform. In der ersten Tabelle sehen Sie die Umsätze des aktuellen und des vergangenen Jahres, in der zweiten Tabelle sehen Sie die Bestellungen des aktuellen und des vergangenen Jahres.

  • Konto

    Hier können Sie verschiedene Einstellungen rund um Ihr lieferando-Partner-Konto vornehmen.


    E-Mail-Adresse verwalten
    Hier können Sie eine Kontaktadresse hinterlegen. Haben sich andere Daten geändert, klicken Sie bitte hier.


    Passwort ändern
    Hier können Sie Ihr Passwort für das lieferando-Partner-Konto ändern. Geben Sie zuerst Ihr altes Passwort ein. Danach müssen Sie nur noch zweimal Ihr neues Passwort eingeben.


    Einstellungen
    Sie können direkt eine Nachricht erhalten, wenn eine Bestellung eingeht, während Sie im Partner-Konto sind. Wählen Sie hier, ob Sie über eine neue Bestellung direkt informiert werden möchten. Wählen Sie außerdem, ob Sie einen Sound hören möchten, wenn eine Bestellung eingeht.

  • Rabatt – Ich möchte einen Rabatt einstellen.

    Klicken Sie hierfür auf den Menüpunkt „Rabatt“ und danach auf „Rabatt hinzufügen“. Nun haben Sie folgende Möglichkeiten:

    • Name: Wählen Sie einen Namen für Ihre Rabattaktion
    • Zeitraum: Legen Sie einen Zeitraum fest, indem Sie auf die Felder hinter „Von“ und „Bis“ klicken. Soll der Rabatt immer gültig sein, klicken Sie auf „Täglich“
    • Rabatt: Wählen Sie danach die Höhe des Rabatts. Stellen Sie ein, ob der Rabatt in € oder in % vom Gesamtpreis sein soll. Klicken Sie auf „Lieferkosten entfallen“, wenn vorhandene Lieferkosten entfallen sollen.
    • Bestellwert: wenn Sie möchten, können Sie hiereinen Mindestbestellwert in € einstellen.
    • Rabatt auf Kategorie: Sie können auch auf einzelne Kategorien Rabatte geben. Möchten Sie nur eine oder auch zwei Kategorien mit einem Rabatt versehen, wählen Sie bitte per Klick aus. Soll der Rabatt doch auf die ganze Karte sein, dann klicken Sie einfach auf „Alle“
    • Wiederholung: hier können Sie einstellen, ob der Rabatt einmalig sein soll, oder ob er sich wiederholen soll. Ein Rabatt kann sich täglich, wöchentlich oder monatlich wiederholen.
  • Wie kann ich lieferando etwas mitteilen? - Kontakt

    Wenn Sie {$config->domain->base} etwas mitteilen möchten, gibt es folgende Möglichkeiten:

    • Anrufen unter {$config->locale->tel->support}
    • Faxen an {$config->locale->fax->support}
    • Oder nutzen Sie das Kontaktformular

    Um das Kontaktformular zu nutzen, schreiben Sie im Partner-Konto unter Kontakt uns einfach eine Nachricht. Wählen Sie dann einen passenden Betreff. Sie können auch eine Datei anhängen. Mit einem Klick auf „Kontaktanfrage schicken“ ist Ihre Nachricht wenige Minuten später bei {$config->domain->base}.
